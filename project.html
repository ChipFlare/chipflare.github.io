CBPaint

CBPaint is a really cool project I made for the CrapBox or really any Arduino with a SSD1306 display attached to it. You can upload the Arduino code it comes with to the Arduino or CrapBox and run the CBPaint Windows app to draw literally anything on the display! You can even upload images.

Github repository: Github

Compiled EXE (No future updates): Google Drive

CB Screen Mirror

Screen mirror does exactly what the name suggests. It mirrors your computer screen to the CrapBox display. Pretty cool when you want to watch a video on less than 144p (actually 64p) for no reason. This comes bundled with the CBPaint application so you can find it in the same repository.

Github repository: Github

Compiled EXE (No future updates): Google Drive

Flappy Windows

It's Flappy Bird, but the windows move! Basically you launch the python script and it will make a bunch of windows that will be obstacles that you have to flap through.

Human readable image format

It's basicaly what the name suggests. you can both view the image in photos and also read it in notepad. Image compression is non existent but the image comes out lossy anyways.
